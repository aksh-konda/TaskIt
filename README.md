# TaskIt

A monorepo containing the TaskIt application — a Spring Boot backend and a React frontend.

## Structure

```
TaskIt/
├── backend/    # Spring Boot REST API (Java, Gradle)
└── frontend/   # React app (Vite)
```

## Getting started

### Backend
```bash
cd backend
./gradlew bootRun
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
